# クッキー型メーカー

手書き画像からクッキー型とスタンプの3Dモデル(STL)を生成するツール

## 入力

- 手書き絵の画像（jpg, jpeg, png）

## 出力

- クッキー型: `{name}.stl`
- スタンプ: `{name}_stamp.stl`

## 処理フロー

### Step 1: 背景削除

- **使用**: Gemini 2.5 Flash (画像生成)
- **処理内容**:
  - 方眼用紙などの背景を削除
  - 背景を純粋な白(#FFFFFF)にする
  - 線はそのまま維持
- **出力**: `{name}_step1_bg_removed.png`

### Step 2: 線の正規化

- **使用**: Gemini 2.5 Flash (画像生成)
- **処理内容**:
  - 全ての線を黒(#000000)に統一
  - 背景は白(#FFFFFF)のまま維持
  - 線のノイズ（端の細いひげ、意図しない飛び出し）を除去
- **出力**: `{name}_step2_normalized.png`

### Step 3: 外枠と内側の分離

- **使用**: Gemini 2.5 Flash (画像生成)
- **処理内容**:
  - 内側の線が外枠に接している部分を切断
  - 切断は内側の線を少し短くして行う（外枠は触らない）
  - 外枠の形状は変更しない
- **出力**: `{name}_step3_separated.png`

### Step 4: クッキー型用枠の抽出

- **使用**: OpenCV
- **処理内容**:
  - `cv2.findContours` で輪郭検出（RETR_EXTERNAL で外側のみ）
  - 最大面積の輪郭を選択（外枠）
  - 輪郭を描画して画像生成
- **出力**: `{name}_step4_contour.png`

### Step 5: スタンプ用画像の生成

- **使用**: OpenCV
- **処理内容**:
  - Step 3の画像からStep 4の輪郭を差し引く
  - 輪郭線を膨張させて確実に消す
  - 内部の要素（文字、模様など）のみを残す
- **出力**: `{name}_step5_stamp_raw.png`

### Step 6: スタンプ用画像の正規化

- **使用**: Gemini 2.5 Flash (画像生成)
- **処理内容**:
  - Step 2と同様の線の正規化処理
- **出力**: `{name}_step5_stamp.png`

### SVG変換

- **使用**: OpenCV
- **処理内容**:
  - グレースケール変換
  - 二値化
  - 輪郭検出
  - SVGファイル書き出し（輪郭の簡略化・スムージング適用）
- **出力**:
  - クッキー型用: `{name}.svg`
  - スタンプ用: `{name}_stamp.svg`

### STL生成

- **使用**: trimesh

#### クッキー型 (`{name}.stl`)

- 抜き型形状
- デフォルト設定:
  - 高さ: 10mm
  - 壁の厚さ: 1mm
  - 目標サイズ: 50mm

#### スタンプ (`{name}_stamp.stl`)

- 構成:
  - 板: クッキー型の輪郭を1mm内側にオフセットした形状
  - 取手: 板の上に配置
  - 凸部分: スタンプ用画像の線を凸として板に配置
- デフォルト設定:
  - 板の厚さ: 3mm
  - 取手サイズ: 10mm x 10mm
  - 取手高さ: 15mm
  - 凸部分の高さ: 1.5mm
  - 凸部分の線幅: 1mm

## 使用方法

```bash
python main.py <画像パス> [オプション]
```

### オプション

| オプション | デフォルト | 説明 |
|-----------|-----------|------|
| `-o, --output` | output | 出力ディレクトリ |
| `--height` | 10.0 | クッキー型の高さ(mm) |
| `--wall` | 1.0 | 壁の厚さ(mm) |
| `--size` | 50.0 | 目標サイズ(mm) |
| `--epsilon` | 0.01 | 輪郭簡略化の係数 |
| `--smoothing` | 0.2 | スムージングの強さ |
| `--no-stl` | - | STL生成をスキップ |
| `--no-stamp` | - | スタンプ生成をスキップ |
| `--no-simplify` | - | 輪郭の簡略化を無効化 |
| `--stamp-plate` | 3.0 | スタンプ板の厚さ(mm) |
| `--stamp-handle-size` | 10.0 | 取手の一辺(mm) |
| `--stamp-handle-height` | 15.0 | 取手の高さ(mm) |
| `--stamp-relief` | 1.5 | 凸部分の高さ(mm) |
| `--stamp-line-width` | 1.0 | 凸部分の線幅(mm) |

## 技術スタック

- **AI画像処理**: Google Gemini 2.5 Flash (画像生成モデル)
- **画像処理**: OpenCV
- **3Dモデル生成**: trimesh
- **実行環境**: Docker
